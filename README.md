## ğŸš€ Quick Start

### Prerequisites
- Python 3.11+
- Node.js 18+
- Conda/Miniconda

### 1. Setup Environment
```bash
# Clone repository
git clone https://github.com/muhammadsaadkhankor/mcr-lab-voice-cloner.git
cd mcr-lab-voice-cloner

# Create conda environment
conda create -n gen_voice python=3.11
conda activate gen_voice
```

### 2. Install Dependencies
```bash
# System dependencies
sudo apt install espeak ffmpeg  # Ubuntu/Debian
brew install espeak ffmpeg      # macOS

# Python packages
pip install -r requirements.txt
pip install -r backend_requirements.txt
pip install PyPDF2
conda install -c conda-forge llama-cpp-python

# Frontend packages
cd frontend && npm install && cd ..
```

### 3. Run Application
```bash
# Terminal 1 - Backend
python app.py

# Terminal 2 - Frontend  
cd frontend && npm start
```

### 4. Access Application
- **Web Interface:** http://localhost:3000
- **API Endpoint:** http://localhost:5000

## ğŸ’» Usage

### Web Interface
1. **Record/Upload Voice** - Use browser recording or upload MP3/WAV
2. **Review Transcript** - Auto-generated by Whisper
3. **Submit Reference** - Confirm your voice sample
4. **Generate Speech** - Enter text to convert to speech
5. **Create API** - Generate API credentials for programmatic access

### API Usage

**Basic Usage (saves in project directory):**
```bash
curl -X POST http://localhost:5000/api/tts \
  -H "Authorization: Bearer mcr_master_api_key_2024" \
  -H "Content-Type: application/json" \
  -d '{
    "voice_id": "voice_professor_abed",
    "text": "Hello, this is my cloned voice!"
  }'
```

**Save to Downloads folder:**
```bash
curl -X POST http://localhost:5000/api/tts \
  -H "Authorization: Bearer mcr_master_api_key_2024" \
  -H "Content-Type: application/json" \
  -d '{
    "voice_id": "voice_professor_abed",
    "text": "Hello, this is my cloned voice!",
    "output_path": "'$HOME'/Downloads/my_voice_output.wav"
  }'
```

**Save to custom path:**
```bash
curl -X POST http://localhost:5000/api/tts \
  -H "Authorization: Bearer mcr_master_api_key_2024" \
  -H "Content-Type: application/json" \
  -d '{
    "voice_id": "voice_tariq_amin",
    "text": "This will be saved to my custom location!",
    "output_path": "/path/to/your/custom/folder/output.wav"
  }'
```

**Available Voice IDs:**
- `voice_professor_abed`
- `voice_tariq_amin` 
- `voice_saad`
